
## 准备工作

```bash
# 进入目录
cd install/compose



# 设置程序运行权限
chmod +x conf/echocloud-admin-web/echocloud-tools.sh  conf/echocloud-admin-web/start.sh  conf/echocloud-admin-web/echocloud-admin-web.jar


```


## 启动服务

```bash
# 方式1.直接启动

docker compose up

# 方式2.后台启动

docker compose up -d

```

浏览器访问 http://localhost
